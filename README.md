# Retail Sales Data Pipeline & Business Intelligence Dashboard

## 📌 Project Overview
This project demonstrates an end-to-end data pipeline built using SQL, Python, and Power BI to analyze retail sales data and generate business insights.

## 🛠 Tools & Technologies
- SQL Server
- Python (pandas, pyodbc, sqlalchemy)
- Power BI

## ⚙️ Pipeline Workflow
1. Extract data from SQL Server
2. Transform data using Python
3. Load cleaned data into CSV
4. Visualize insights in Power BI

## 📊 Key Insights
- Technology category generates highest sales
- West region contributes the most revenue
- Sales show fluctuating but overall upward trend

## 📁 Project Structure
- scripts/ → ETL pipeline
- data/ → cleaned dataset
- dashboard/ → Power BI dashboard

## 🚀 Outcome
Built a complete data pipeline and dashboard for business decision-making.
